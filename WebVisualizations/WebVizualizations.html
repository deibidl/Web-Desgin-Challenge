Latitude

What happens when you get closer to the equator.

SUMMARY

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of whether vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This sire provides the source data and the visualizations created as part of the analysis, as well as the explanations and descriptions of any trends and collections witneseed
